=================

   *  -  Field

   *  -  **nodes** |br|
         array of objects
      -

      -

         object
      -
      -  The parent :doc:`ValueTreeNode` object.
      -

         string (GUID)
      -
      -  The id of the object.
      -
   *  -  **text** |br|
         string
      -
      -  The display text of the object
      -
   *  -  **value** |br|
         string
      -
      -  The value of the object
